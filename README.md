# 0705-GoToElement
0705-滾動至指定區域
展示頁面: https://simpsonyuki.github.io/0705-GoToElement/

使用步驟:

STEP1 / CDN GoToElement.js
```
<script src="https://simpsonyuki.github.io/0705-GoToElement//GoToElement.js"></script>
```

STEP2 / 添加屬性
```
data-gte-target="目標ID名稱" 

data-gte-offset="位移(上方空間)"
```

STEP3 / 添加目標ID
```
<div id="目標ID"></div>
```
